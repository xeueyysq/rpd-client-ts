Конструктор РПД — документация

Руководство пользователя по работе с системой формирования рабочих программ дисциплин.

Введение

«Конструктор РПД» — это веб-приложение для создания, согласования и выгрузки рабочих программ дисциплин. Документация открывается из сайдбара по ссылке Документация и доступна на всех страницах приложения.

Роли

  • РОП / Администратор — создание комплектов, управление шаблонами, компетенциями и пользователями.
  • Преподаватель — заполнение шаблонов РПД по закреплённым дисциплинам.

Комплекты РПД

Раздел «Создание комплекта» позволяет сформировать новый комплект рабочих программ, а «Список комплектов» — открыть ранее созданные комплекты для просмотра и редактирования.

Шаблоны

Преподаватели работают с шаблонами РПД через раздел «Список шаблонов». Каждый шаблон содержит разделы программы, которые заполняются и сохраняются автоматически.

Компетенции

Раздел «Список компетенций» содержит планируемые результаты обучения, которые привязываются к дисциплинам при формировании РПД.

Частые вопросы

Как выгрузить РПД в Word? — Выгрузка доступна со страницы комплекта; документ формируется в формате .docx.

Не сохраняются изменения. — Проверьте подключение к сети и обновите страницу; данные сохраняются автоматически при переходе между разделами.
